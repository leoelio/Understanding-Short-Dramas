## 本次变更摘要

- 仅在 短剧理解-android / native-android-migration 工作，未启动 Web 服务端口，未修改 Web worktree。
- 聊聊页新增 /api/watch-rooms/invitations 数据加载，并把待处理同看邀请计入社交状态。
- 同看邀请卡片展示收到的邀请、发出的邀请、房间码、episode_id 和状态。

- 本轮暂不实现完整房间同步播放页，只完成邀请收发处理闭环。
