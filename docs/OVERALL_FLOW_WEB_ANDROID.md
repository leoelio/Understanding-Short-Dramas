## 飞书使用方法

1. 打开飞书文档。
2. 输入 `/图片`，或直接把 `docs/overall_flow_web_android_feishu.png` 拖进飞书文档。
3. 如果图片过大，点击图片后选择“适应页面宽度”。
4. Mermaid 代码只作为备份，不建议直接粘贴到飞书正文中。

## Mermaid 备份版本

```mermaid
flowchart TD
    A["短剧素材库<br/>视频、封面、字幕、弹幕 CSV"]
    B["内容导入与预处理<br/>扫描剧集、生成基础数据"]
    C["大模型离线理解<br/>高光表达、贴图策略、弹幕候选、二创分镜"]
    D["人工复核工作台<br/>确认时间点、内容安全和展示质量"]
    E["FastAPI 服务端<br/>统一接口层 / 统计 / 复核"]
    DB["SQLite 数据库<br/>后续迁移 PostgreSQL"]
    AS["资产缓存<br/>视频、贴图、图片、mp3"]
    API["统一 API 层<br/>短剧 / 剧集 / 高光 / 弹幕 / 互动 / 社交 / 声音 / 二创"]

    W["Web 主线客户端<br/>当前展示、答辩、复核和完整体验主版本"]
    W1["登录 / 选片首页"]
    W2["Web 播放页"]
    W3["高光互动<br/>弹层 / 贴图 / 狂点"]
    W4["弹幕三档<br/>轻聊 / 狂欢 / 沉浸"]
    W5["片尾 AI 二创<br/>分支 / 图片 / 声音"]

    N["Android 支线客户端<br/>验证原生播放、全屏、触摸、权限和 App 化"]
    N1["原生登录 / 选剧"]
    N2["原生播放页"]
    N3["接入高光时间轴"]

    V1["用户授权声音样本<br/>我的页录音 / 上传"]
    V2["Voice Profile<br/>保存授权和样本信息"]
    V3["CosyVoice 生成 mp3<br/>预设文本生成并缓存"]

    G["弹幕治理七层链路<br/>规则层 → 时间感知 → 语义分类 → 聚类去重 → 小模型 → 大模型复审候选 → 人工复核"]
    F1["用户互动数据回流<br/>点击、选择、弹幕、二创、同看"]
    F2["后台统计与策略优化<br/>高光效果、弹幕治理、二创内容"]

    A --> B --> C --> D --> E
    E --> DB
    E --> AS
    E --> API
    API --> W
    API --> N

    W --> W1 --> W2 --> W3 --> W4 --> W5
    N --> N1 --> N2 --> N3

    V1 --> V2 --> V3 --> W5
    G --> W4
    W --> F1 --> F2 --> D
```
