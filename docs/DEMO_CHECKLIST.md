# 答辩演示检查清单

更新时间：2026-06-11

## 文档定位

这份清单用于答辩、录屏或临时演示前快速确认“半句”能稳定展示。它不是开发说明，而是演示前的操作检查表。

## 1. 基础服务

- [ ] 当前分支确认在 Web 主线，例如 `web-product`。
- [ ] 后端服务已启动：

```powershell
.\.venv\Scripts\python.exe -m uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```

- [ ] 本地首页能打开：`http://127.0.0.1:8000/`
- [ ] API 自动文档能打开：`http://127.0.0.1:8000/docs`
- [ ] 健康检查正常：`GET /api/health`
- [ ] 浏览器控制台没有持续刷新的红色错误。

## 2. 数据和素材

- [ ] `data/app.db` 存在。
- [ ] `视频库/` 存在，且剧集视频文件可访问。
- [ ] `/api/dramas` 能返回短剧列表。
- [ ] `/api/episodes/3` 能返回《北往》第 1 集详情。
- [ ] `/media/episodes/3` 能播放视频。
- [ ] 头像池和封面图能正常显示。
- [ ] `frontend/assets/stickers/` 中贴纸资源可访问。
- [ ] `frontend/assets/companion/xiaoban_3d_v1.png` 可访问。

## 3. 登录和用户

- [ ] 登录页文字清晰，按钮可点击。
- [ ] 能使用演示账号登录。
- [ ] 登录后进入短剧首页。
- [ ] “我的”页面能展示头像、昵称、成长信息和声音资产入口。
- [ ] 不在录屏中展示任何真实密钥或本地私密路径。

## 4. 播放页核心体验

推荐演示路径：

```text
http://127.0.0.1:8000/?episode=3
```

检查项：

- [ ] 进入播放页后页面自动回到顶部。
- [ ] 顶部导航和标题区不干扰观看。
- [ ] 视频可播放、暂停、拖动进度。
- [ ] 鼠标不动时控制栏会隐藏。
- [ ] 鼠标移动后控制栏会出现。
- [ ] 进度条悬停能显示时间提示。
- [ ] 全屏按钮可用。
- [ ] 退出或离开播放页后视频停止播放。

## 5. 高光和贴图

- [ ] 高光弹层不会遮挡剧情中心。
- [ ] 高光互动按钮清晰，点击后有反馈。
- [ ] 疯狂点击或连续点击类效果不会无限遮挡。
- [ ] 贴图出现位置有变化，不全部固定在同一处。
- [ ] 贴图频率能体现情绪，但不会把整集盖满。
- [ ] Q 版进度标记和底部进度线同步。

20 集 LLM 升级检查：

- [ ] `data/llm_upgrades/summary_all.json` 存在。
- [ ] `data/llm_upgrades/episode_*_highlight_sticker_upgrade.json` 覆盖 20 集。
- [ ] 复核页能看到 `llm_draft` 体验配置。
- [ ] 抽查至少 2-3 集：高光文案、分类、按钮和贴图时间窗没有明显离题。

## 6. 弹幕治理

- [ ] 弹幕三档模式可切换：轻聊、狂欢、沉浸。
- [ ] 弹幕能按时间出现。
- [ ] 弹幕点击、点赞、回复入口可用。
- [ ] 复核页能打开弹幕治理列表。
- [ ] 弹幕治理摘要能展示七层链路：规则、时间感知、语义分类、聚类去重、小模型、大模型复审候选、人工复核。
- [ ] 小模型版本、训练行数、大模型复审候选数量能展示。
- [ ] 如需演示训练，使用后台接口或复核页按钮触发，不在现场展示密钥。

## 7. 小半陪看和片尾战报

- [ ] 播放中“小半”提示气泡能出现和自动消失。
- [ ] 提示内容短，不遮挡主要剧情。
- [ ] 用户点击高光后，片尾战报能统计点击、选择和奖励。
- [ ] 片尾战报能展示“小半”形象。
- [ ] 3D 形象和灵动眼睛模式可切换。

## 8. 片尾 AI 二创

推荐演示路径：

```text
http://127.0.0.1:8000/?episode=3&remix=1
```

检查项：

- [ ] 正片结束后才出现片尾 AI 二创入口。
- [ ] 入口不会和最后一个高光弹层抢展示时机。
- [ ] 三条主分支可点击。
- [ ] 分支内个性化选项能依次出现。
- [ ] 图片分镜能铺满主要画面，不被裁掉一半。
- [ ] 点击图片或等待后能进入下一张。
- [ ] 原版声音和用户声音入口分开。
- [ ] 同一时间只播放一段声音，不重叠。
- [ ] 如果声音服务不可用，页面要有可理解的提示，不阻断图片分镜展示。

## 9. 声音资产

- [ ] 本地 CosyVoice 服务按需启动。
- [ ] `COSYVOICE_BASE_URL` 指向本地服务。
- [ ] “我的”页面能上传或录入声音样本。
- [ ] 授权文本为：`同意利用录入声音生成音频`。
- [ ] 生成试听能返回 mp3。
- [ ] 已生成音频优先走缓存，不重复生成。
- [ ] 录屏中不展示用户私密声音文件路径。

## 10. 同看和社交

- [ ] 好友申请、接受、拒绝、撤回和历史记录可展示。
- [ ] 聊聊页面以会话为主，不只是邀请好友。
- [ ] 能发送文字、表情或同看链接。
- [ ] 同看房间可创建。
- [ ] 房间成员卡能展示头像、昵称、称号、徽章。
- [ ] 演示模式可注入对方答题、弹幕、点赞和高光选择事件。
- [ ] 逛逛动态流能展示文字感受和 AI 资产类型预留。

## 11. 后台复核

打开：

```text
http://127.0.0.1:8000/#review
```

检查项：

- [ ] 剧集列表可筛选。
- [ ] 单集高光可查看和编辑。
- [ ] 高光时间可通过输入数字调整。
- [ ] 体验配置可查看和保存。
- [ ] 贴图时间窗可新增、删除、预览和导入。
- [ ] 弹幕治理可筛选、复核和重跑。
- [ ] 片尾二创记录可设为精选或隐藏。

## 12. 公网演示

如果需要给不在同一网络的人看：

- [ ] 本地 8000 服务已启动。
- [ ] 启动临时公网隧道：

```powershell
.\scripts\start_public_tunnel.ps1
```

- [ ] 复制脚本输出的 HTTPS 地址。
- [ ] 用手机或另一台电脑打开公网地址检查首页、视频和资源。
- [ ] 演示结束后停止隧道：

```powershell
.\scripts\stop_public_tunnel.ps1
```

注意：临时隧道地址不稳定，重启后通常会变化。

## 13. 录屏建议

建议录屏顺序：

1. 登录页。
2. 短剧首页和最近观看。
3. 《北往》第 1 集播放页。
4. 高光互动和贴图。
5. 弹幕模式和治理说明。
6. 小半陪看提示。
7. 片尾 AI 二创。
8. 原版/用户声音播放。
9. 同看房间社交展示。
10. 个人主页和勋章展馆。
11. 复核页展示高光、贴图、弹幕治理和二创精选。

## 14. 演示风险和兜底

| 风险 | 兜底 |
| --- | --- |
| 大模型接口不可用 | 使用已缓存高光、贴图和二创结果 |
| 声音服务不可用 | 展示图片分镜和已有缓存 mp3 |
| 公网隧道不稳定 | 本地录屏或现场同机演示 |
| 视频加载慢 | 提前打开目标剧集，确认缓存 |
| 贴图或弹层遮挡 | 演示前在复核页调整时间窗 |
| 账号状态异常 | 使用本地默认演示账号或重新注册 |

## 15. 演示前最后动作

- [ ] 关闭无关窗口和敏感信息。
- [ ] 清理浏览器地址栏中的无关参数。
- [ ] 确认音量和麦克风状态。
- [ ] 准备好答辩讲稿。
- [ ] 不现场生成不可控视频。
- [ ] 不展示真实 API Key。
